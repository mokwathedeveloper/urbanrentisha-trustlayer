# UrbanRentisha TrustLayer Tenant Onboarding Page Starter

This starter implements **Screen 3: Tenant Onboarding Page only**.

## Purpose

The page explains how verified viewing works and why payment proof is required before access unlocks.

## Includes

```text
Verified viewing explanation
Payment proof explanation
ZK + Stellar trust flow
Step-by-step onboarding cards
Access unlock rules
Progress stepper
Interactive onboarding state
CTA to browse verified properties
CTA to try demo request
Google Fonts
TypeScript components
Tailwind CSS tokens
```

## Route

```text
/tenant/onboarding
```

## Run

```bash
pnpm install
pnpm dev
```
