# UrbanRentisha TrustLayer Signup / Login Page Starter

This starter implements **Screen 2: Signup / Login Page only**.

## Includes

```text
Demo login
Email login
Signup mode
Wallet-based access
Trust/security side panel
Role selector
Accessible inputs
Google Fonts
TypeScript components
Tailwind CSS tokens
```

## Stack

```text
Next.js App Router
TypeScript
Tailwind CSS
Google Fonts: Inter + JetBrains Mono
Lucide React
```

## Run

```bash
pnpm install
pnpm dev
```

## Route

```text
/auth
```
