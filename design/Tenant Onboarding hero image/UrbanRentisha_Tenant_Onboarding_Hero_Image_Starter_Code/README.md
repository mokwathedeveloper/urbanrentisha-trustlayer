# UrbanRentisha Tenant Onboarding Hero Illustration Starter

This starter shows how to use the right-side Tenant Onboarding hero illustration in the onboarding page.

## Image filename

Place the image here:

```text
public/images/tenant-onboarding-hero-illustration.png
```

## Recommended source image dimension

```text
1600px × 1100px
Aspect ratio: about 16:11
Format: PNG
Background: dark/transparent-style visual
```

## Website display size

```text
Desktop: 640px × 440px
Tablet: 560px × 385px
Mobile: 100% width, about 320px–360px height
```

## Run

```bash
pnpm install
pnpm dev
```
