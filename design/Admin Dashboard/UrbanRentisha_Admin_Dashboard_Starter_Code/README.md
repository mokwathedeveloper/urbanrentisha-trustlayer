# UrbanRentisha TrustLayer Admin Dashboard Starter

This starter implements **Screen 18: Admin Dashboard only**.

## Purpose

Shows platform-wide approvals, reports, agents, proof verification activity, suspicious activity, audit logs, and analytics.

## Route

```text
/admin/dashboard
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Admin overview
Platform-wide approvals
Reports center
Agent management
Proof verification activity

Audit logs

Responsive layout
```
