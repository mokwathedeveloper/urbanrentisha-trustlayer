# UrbanRentisha TrustLayer Proof Verification Screen Starter

This starter implements **Screen 10: Proof Verification Screen only**.

## Purpose

Submits the generated proof for Stellar/Soroban verification and shows proof verification status.

## Route

```text
/proof-verification/REQ-UR-9084
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Generated proof summary
Proof submission action
Stellar/Soroban verification flow
Verification status states
Verification hash output
Contract/verifier reference
Public input review
Audit event panel
Access unlock explanation
Success CTA to viewing code/access unlock
Google Fonts
TypeScript components
Tailwind CSS tokens
```
