# UrbanRentisha TrustLayer Request Viewing Screen Starter

This starter implements **Screen 7: Request Viewing Screen only**.

## Purpose

Creates a viewing request for a selected verified property and shows the required viewing fee before the tenant proceeds to Stellar payment and ZK proof verification.

## Route

```text
/request-viewing/UR-LST-1001
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Selected property summary
Required viewing fee
Viewing request form
Preferred viewing date and time
Contact preference selector
Tenant notes textarea
Payment-proof explanation
Fee breakdown
Safety and agent verification panel
Submit request interaction
Success state with request ID
Next step CTA to Stellar payment
TypeScript components
Google Fonts
Tailwind CSS tokens
```
