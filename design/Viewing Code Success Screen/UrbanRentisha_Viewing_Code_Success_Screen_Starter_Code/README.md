# UrbanRentisha TrustLayer Viewing Code Success Screen Starter

This starter implements **Viewing Code Success Screen only**.

## Purpose

Displays the unlocked viewing code after successful proof verification.

## Route

```text
/viewing-code/REQ-UR-9084
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Unlocked viewing code
Copy code interaction
Show / hide code interaction

Viewing schedule

Google Fonts
TypeScript components
```
