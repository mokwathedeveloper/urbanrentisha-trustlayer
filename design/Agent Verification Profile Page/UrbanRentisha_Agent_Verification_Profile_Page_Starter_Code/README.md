# UrbanRentisha TrustLayer Agent Verification Profile Page Starter

This starter implements **Screen 15: Agent Verification Profile Page only**.

## Purpose

Shows agent verification status, trust score, report count, listed properties, and verified viewing requests.

## Route

```text
/agents/AGT-UR-2048
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Agent profile header
Verification status
Trust score
Report count
Listed properties
Verified viewing requests
Agent verification documents
Trust timeline
Safety and report summary
Contact/actions panel
Google Fonts
TypeScript components
Tailwind CSS tokens
```
