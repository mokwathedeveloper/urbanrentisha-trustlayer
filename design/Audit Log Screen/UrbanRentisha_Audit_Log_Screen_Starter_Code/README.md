# UrbanRentisha TrustLayer Audit Log Screen Starter

This starter implements **Screen 19: Audit Log Screen only**.

## Purpose

Shows important system events and trust activity across UrbanRentisha TrustLayer.

## Route

```text
/audit-log
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Audit log hero
Audit statistics
Search and filter bar
System event timeline
Trust activity feed
Audit event table
Selected event details panel
Export controls
Integrity status card
Sidebar navigation
Google Fonts
TypeScript components
Tailwind CSS tokens
Responsive layout
```
