# UrbanRentisha TrustLayer Property Manager Dashboard Starter

This starter implements **Screen 17: Property Manager Dashboard only**.

## Purpose

Shows manager listings, viewing requests, verified tenants, escrow/payment-hold statuses, reports, viewing codes, and trust score.

## Route

```text
/manager/dashboard
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Property manager dashboard overview
Manager listing performance
Viewing request queue
Verified tenant panel
Escrow / payment-hold statuses
Viewing code management
Reports center
Manager trust score
Quick actions
Recent activity
Audit references
Google Fonts
TypeScript components
Tailwind CSS tokens
Responsive layout
```
