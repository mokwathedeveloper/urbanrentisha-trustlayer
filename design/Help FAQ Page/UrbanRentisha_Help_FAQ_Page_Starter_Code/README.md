# UrbanRentisha TrustLayer Help / FAQ Page Starter

This starter implements **Screen 22: Help / FAQ Page only**.

## Purpose

Explains Stellar testnet, ZK proof, payment-hold status, viewing codes, reports, safety rules, and known limitations.

## Route

```text
/help
```

## Run

```bash
pnpm install
pnpm dev
```
