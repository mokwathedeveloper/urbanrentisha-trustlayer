# UrbanRentisha TrustLayer API Documentation Page Starter

This starter implements **Screen 20: API Documentation Page only**.

## Purpose

Shows API endpoints for external rental-platform integrations.

## Route

```text
/api-docs
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
API documentation hero
Integration overview
Authentication panel
API endpoint catalog
Request and response examples
Webhook events
SDK cards
External rental-platform integration flow
Rate limits and security notes
API playground preview
Sidebar navigation
Google Fonts
TypeScript components
Tailwind CSS tokens
Responsive layout
```
