# UrbanRentisha TrustLayer Landing Page Starter Code

This is the starter code for the **Landing Page only**.

## Stack

```text
Next.js App Router
TypeScript
Tailwind CSS
Google Fonts
Lucide React Icons
```

## Install dependencies

```bash
pnpm install
pnpm add lucide-react clsx tailwind-merge
```

## Run

```bash
pnpm dev
```

## Landing page sections included

```text
LandingNavbar
HeroSection
TrustStrip
ProblemSection
SolutionFlowSection

HowItWorksSection

ApiIntegrationSection

LandingFooter
```
