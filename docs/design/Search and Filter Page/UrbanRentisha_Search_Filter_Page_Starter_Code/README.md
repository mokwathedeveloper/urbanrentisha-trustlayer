# UrbanRentisha TrustLayer Search and Filter Page Starter

This starter implements **Screen 6: Search and Filter Page only**.

## Purpose

Allows tenants to filter verified properties by location, rent range, verification status, property type, and viewing fee.

## Route

```text
/search
```

## Run

```bash
pnpm install
pnpm dev
```

## Included

```text
Search input
Location filter
Rent range filter
Viewing fee filter
Verification status filter
Property type filter
Bedroom filter
Active filter chips
Sort control
Verified result cards
Saved search panel
Mobile filter drawer
Empty state
TypeScript components
Google Fonts
Tailwind CSS tokens
```
