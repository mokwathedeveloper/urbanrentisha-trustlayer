# UrbanRentisha TrustLayer Tenant Dashboard Starter

This starter implements **Screen 16: Tenant Dashboard only**.

## Purpose

Shows tenant requests, payment statuses, proof statuses, escrow/payment-hold statuses, viewing codes, notifications, reports, and viewing history.

## Route

```text
/tenant/dashboard
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Tenant dashboard overview
Active viewing request cards
Payment status tracking
Proof status tracking
Escrow/payment-hold status tracking
Viewing code status
Notifications summary
Report summary
Viewing history
Quick actions
Trust flow timeline
Google Fonts
TypeScript components
Tailwind CSS tokens
Responsive layout
```
