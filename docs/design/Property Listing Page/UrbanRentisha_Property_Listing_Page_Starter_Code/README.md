# UrbanRentisha TrustLayer Property Listing Page Starter

This starter implements **Screen 4: Property Listing Page only**.

## Purpose
Shows verified rental listings with property cards, search, filters, and request-viewing buttons.

## Includes
```text
Verified rental listing grid
Search input
Location filter
Property type filter
Price filter
Bedroom filter
Verified-only toggle
Sort control
Saved property interaction
Request viewing side panel
Trust score badges
Demo property data
Google Fonts
TypeScript components
Tailwind CSS tokens
```

## Route
```text
/listings
```

## Run
```bash
pnpm install
pnpm dev
```
