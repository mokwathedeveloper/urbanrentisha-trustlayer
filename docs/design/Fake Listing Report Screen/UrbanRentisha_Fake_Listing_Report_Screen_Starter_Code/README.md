# UrbanRentisha TrustLayer Fake Listing Report Screen Starter

This starter implements **Screen 14: Fake Listing Report Screen only**.

## Purpose

Allows tenants to report suspicious listings or agents.

## Route

```text
/report-listing/UR-LST-1001
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Suspicious listing report form
Report type selector
Suspicious reason selector
Severity selector
Property summary card
Agent summary card
Evidence upload UI
Reporter privacy options
Safety warning panel
Review process timeline
Submission success state
Report reference ID
Google Fonts
TypeScript components
Tailwind CSS tokens
```

## UX rule

This screen must feel serious, safe, and calm. It should help tenants report fraud without fear, while avoiding exaggerated claims or legal overpromising.
