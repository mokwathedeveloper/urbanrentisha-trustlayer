# UrbanRentisha Stellar Banner Mark Starter

This starter shows how to use the Stellar banner image inside the UrbanRentisha landing page banner section.

## Image requirement

Place the image here:

```text
public/images/stellar-banner-mark.png
```

Recommended export size:

```text
720px × 240px
Aspect ratio: 3:1
Format: PNG
Background: Transparent
```

## Run

```bash
pnpm install
pnpm dev
```
