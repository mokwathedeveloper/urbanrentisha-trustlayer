# UrbanRentisha TrustLayer Landing Page Starter

This starter contains the Landing Page UI/UX blueprint and TypeScript component code for UrbanRentisha TrustLayer.

## Stack

```text
Next.js App Router
TypeScript
Tailwind CSS
Google Fonts: Inter + JetBrains Mono
Lucide React Icons
```

## Install

```bash
pnpm add lucide-react clsx tailwind-merge
```

## Use

Copy the files into:

```text
apps/web/
```

Then run:

```bash
pnpm dev
```
