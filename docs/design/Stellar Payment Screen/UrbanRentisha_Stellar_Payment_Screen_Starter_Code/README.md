# UrbanRentisha TrustLayer Stellar Payment Screen Starter

This starter implements **Screen 8: Stellar Payment Screen only**.

Shows the Stellar testnet payment flow, payment amount, transaction status, and transaction hash.

## Route

```text
/stellar-payment/REQ-UR-9084
```

```bash
pnpm install
```
