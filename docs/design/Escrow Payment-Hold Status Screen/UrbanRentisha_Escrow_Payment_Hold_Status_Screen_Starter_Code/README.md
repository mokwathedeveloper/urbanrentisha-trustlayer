# UrbanRentisha TrustLayer Escrow / Payment-Hold Status Screen Starter

This starter implements **Screen 11: Escrow / Payment-Hold Status Screen only**.

## Purpose

Shows simplified payment-hold or reservation status for a viewing request.

## Route

```text
/payment-hold/REQ-UR-9084
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Simplified payment-hold status
Reservation status
Held viewing fee amount
Property and request summary
Payment-hold timeline
Release/refund rules
Stellar/Soroban reference
Audit events
Tenant action buttons
Help/dispute support panel
Copy hold reference interaction
Google Fonts
TypeScript components
Tailwind CSS tokens
```

## Important product wording

For the hackathon MVP, this screen uses **payment-hold** and **reservation status** wording. Use the word **escrow** carefully unless the final legal/financial model supports true regulated escrow.
