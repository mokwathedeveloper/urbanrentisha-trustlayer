# UrbanRentisha TrustLayer Landing Page Hero Starter

This starter implements the landing page hero section using the provided hero image.

## Important filename

The hero image is already placed here:

```text
public/images/hero-ilustration.png
```

The code references it as:

```text
/images/hero-ilustration.png
```

## Stack

```text
Next.js App Router
TypeScript
Tailwind CSS
Google Fonts: Inter + JetBrains Mono
next/image
Lucide React Icons
```

## Run

```bash
pnpm install
pnpm dev
```
