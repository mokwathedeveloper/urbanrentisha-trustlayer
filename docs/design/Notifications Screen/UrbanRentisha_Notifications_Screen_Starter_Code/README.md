# UrbanRentisha TrustLayer Notifications Screen Starter

This starter implements **Screen 13: Notifications Screen only**.

## Purpose

Shows payment, proof, access, report, and viewing-code updates.

## Route

```text
/notifications
```

## Run

```bash
pnpm install
pnpm dev
```

## Includes

```text
Notification inbox
Payment notifications
Proof notifications
Access notifications
Report notifications
Viewing-code notifications
Unread / read states
Category filter tabs
Search field
Priority badges
Notification detail panel
Mark as read interaction
Mark all as read interaction
Notification preferences panel
Google Fonts
TypeScript components
Tailwind CSS tokens
```
