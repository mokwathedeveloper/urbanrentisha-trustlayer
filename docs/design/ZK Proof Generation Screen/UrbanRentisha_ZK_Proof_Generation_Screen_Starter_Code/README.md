# UrbanRentisha TrustLayer ZK Proof Generation Screen Starter

This starter implements **Screen 9: ZK Proof Generation Screen only**.

## Purpose

Allows the tenant to generate a private payment proof after payment is received.

## Route

```text
/zk-proof-generation/REQ-UR-9084
```

## Run

```bash
pnpm install
pnpm dev
```
