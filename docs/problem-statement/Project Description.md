# Project Description

UrbanRentisha TrustLayer is a privacy-preserving rental trust infrastructure layer built on Stellar. It helps rental platforms, property agencies, landlords, and tenants reduce rental scams by verifying that a tenant has completed a required viewing or reservation payment before private property access details are released.

The system allows a tenant to select a verified rental listing, request viewing access, make a Stellar testnet payment, generate a zero-knowledge proof, and submit that proof to a Stellar Soroban smart contract for verification. Once the proof is verified, the tenant receives a unique viewing code or controlled access to agent contact details.

UrbanRentisha TrustLayer uses zero-knowledge technology to prove that a payment condition has been satisfied without exposing unnecessary wallet information, full transaction history, or unrelated financial activity. This makes the product useful for real-world rental marketplaces where tenants need protection from fake agents and property managers need a reliable way to confirm serious viewing requests.

The MVP includes verified property listings, tenant accounts, viewing requests, Stellar testnet payment, ZK proof generation, Soroban proof verification, viewing code unlock, fake listing reports, agent trust profiles, escrow status tracking, dashboards, audit logs, and demo mode for hackathon judges.

UrbanRentisha TrustLayer is not designed as a full rental marketplace. It is designed as a reusable B2B SaaS trust layer that can be integrated into existing rental platforms to make rental access safer, more transparent, and more verifiable.
